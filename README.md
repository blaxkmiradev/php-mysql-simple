# Simple PHP MySQL App

## 📦 Requirements
- PHP 7+
- MySQL
- XAMPP / Laragon

## ⚙️ Setup

1. Clone repo:
git clone https://github.com/blaxkmiradev/php-mysql-simple.git

2. Import database:
- Open phpMyAdmin
- Import file: sql/database.sql

3. Configure DB:
Edit config/db.php if needed

## ▶️ Run

Move project into:
htdocs (XAMPP)

Open:
http://localhost/php-mysql-simple/public/

## ✅ Features
- Connect to MySQL
- Fetch and display users
